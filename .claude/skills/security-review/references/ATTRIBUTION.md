## Files in this directory

- `asvs-5.0.0.txt` — a plain-text rendering of the standard's official
  machine-readable CSV (`5.0/docs_en`), downloaded from the official release
  <https://github.com/OWASP/ASVS/releases/tag/v5.0.0> and grouped by
  chapter/section so agents can `grep` the requirements. The requirement text is
  reproduced verbatim from the CSV; it is a mechanical reproduction of the same
  CC BY-SA 4.0 material and carries the same licence and attribution. (This repo
  uses the CSV rather than the PDF; the content is identical requirement text.)
- `asvs-map.md` — a derived work: this repository's applicability map against
  ASVS 5.0.0 for a browser SPA. It paraphrases requirements rather than
  reproducing them; as an adaptation of CC BY-SA 4.0 material it is likewise
  shared under CC BY-SA 4.0.
